# InvestaVisor

#### Video Demo: https://youtu.be/dkTXygwaMgs

## Description
InvestaVisor is an educational investment utility that generates structured daily
investing plans based on a small set of user inputs. The application removes the
need for AI prompting and helps users maintain discipline and clarity.

## Files
- app.py: Flask application and API routes
- models/: Database access logic
- services/: Business and AI-related logic
- templates/: HTML templates
- static/: CSS and JavaScript assets
- schema.sql: Database schema

## Design
The application is designed as a lightweight, single-page web app with a clear
separation between presentation, business logic, and persistence layers.

## Disclaimer
InvestaVisor is not a financial advisor and is for educational purposes only.
